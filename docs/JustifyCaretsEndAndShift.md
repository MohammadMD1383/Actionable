## Justify Carets End and Shift Text

Aligns carets to the rightmost one, and also shifts the text with moved carets

---

### example:

| symbol              | meaning |
| ------------------- | ------- |
| <code>&#124;</code> | caret   |

having:
```java
String s |= "some string";
String someString |= "some another string";
```
will produce:
```java
String s          |= "some string";
String someString |= "some another string";
```
