## Justify Carets Start

Aligns the carets to the leftmost one

---

### example:

| symbol              | meaning |
| ------------------- | ------- |
| <code>&#124;</code> | caret   |

having:
```java
class |A {
  final int |a = 122;
  String |str = null;
}
```
will produce:
```java
class |A {
  fina|l int a = 122;
  Stri|ng str = null;
}
```
