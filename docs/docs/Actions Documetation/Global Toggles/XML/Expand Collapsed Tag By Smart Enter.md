Expands a collapsed tag when caret is at the end of the tag and _complete current statement_ happens.

## Example

| Symbol | Meaning |
|:------:|:-------:|
| &vert; |  Caret  |

### Before

```xml

<tag/>|
```

### Action

Press ++ctrl+shift+enter++ _(Complete Current Statement)_

### After

```xml

<tag>|</tag>
```
