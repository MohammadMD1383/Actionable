Smartly helps you to collapse empty `XML` tags by just pressing backspace.

## How?

| Symbol | Meaning |
|:------:|:-------:|
| &vert; |  Caret  |

### Before

```xml

<parent>
	<empty-child>|</empty-child>
</parent>
```

### Action

++back++

### After

```xml

<parent>
	<empty-child/>|
</parent>
```
