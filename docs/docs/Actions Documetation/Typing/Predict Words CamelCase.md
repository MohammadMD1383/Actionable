# Predict Words CamelCase

## What Does It Do?

Predicts words inside a long text.

## Example

| Symbol |  Meaning  |
|:------:|:---------:|
| &vert; |   Caret   |
|  [ ]   | Selection |

### Before

```javascript
const myVar = "[alargecontinousstring]|";
```

### After

```javascript
const myVar = "[aLargeContinousString]|";
```
