# Predict Words SnakeCase

## What Does It Do?

Predicts words inside a long text.

## Example

| Symbol |  Meaning  |
|:------:|:---------:|
| &vert; |   Caret   |
|  [ ]   | Selection |

### Before

```javascript
const myVar = "[alargecontinousstring]|";
```

### After

```javascript
const myVar = "[a_large_continous_string]|";
```
