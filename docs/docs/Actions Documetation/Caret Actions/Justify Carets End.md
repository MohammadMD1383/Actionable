Aligns the carets at the rightmost column.

## Example

| Symbol | Meaning |
|:------:|:-------:|
| &vert; |  Caret  |

### Before

[//]: # (@formatter:off)
```java
String longtext = "longtext";|
String verylongtext = "verylongtext";|
String veryverylongtext = "veryverylongtext";|
```
[//]: # (@formatter:on)

### After

[//]: # (@formatter:off)
```java
String longtext = "longtext";                |
String verylongtext = "verylongtext";        |
String veryverylongtext = "veryverylongtext";|
```
[//]: # (@formatter:on)
