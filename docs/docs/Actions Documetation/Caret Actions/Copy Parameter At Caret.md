Copies the function parameter under the caret.

## Example

[//]: # (@formatter:off)
```javascript
function hello(param1, param2) {
               ^^^^^^  ^^^^^^
	// do somrthing
}
```
[//]: # (@formatter:on)

The specified parts are eligible to be copied.

## Remarks

[//]: # (@formatter:off)
!!! note
    Supported languages:

    * java
    * kotlin
    * javascript
    * typescript
    * golang
[//]: # (@formatter:on)
