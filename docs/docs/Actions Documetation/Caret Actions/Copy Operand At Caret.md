Copies a binary expression operand under the caret.

## Example

[//]: # (@formatter:off)
```kotlin
val a = "some string" + 1334 + "another piece of string"
        ^^^^^^^^^^^^^   ^^^^   ^^^^^^^^^^^^^^^^^^^^^^^^^
```
[//]: # (@formatter:on)

The specified parts are the operands.
