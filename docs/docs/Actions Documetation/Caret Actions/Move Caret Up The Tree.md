Moves the caret up the ==Psi Tree==.

## Example

| Symbol | Meaning |
|:------:|:-------:|
| &vert; |  Caret  |

[//]: # (@formatter:off)
```xml

<component1>
	<component2>
		<component3>
			hello world|
		</component3>
	</component2>
</component1>
```

<component1>
