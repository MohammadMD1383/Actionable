Moves the caret to the next same ==PsiElement==.

## Example

| Symbol |  Meaning  |
|:------:|:---------:|
| &vert; |   Caret   |
|  [ ]   | Selection |

```javascript
const obj = {
	'|': '',
	'': '',
	'': '',
	'': '',
	'': '',
}
```

```javascript
const obj = {
	'': '|',
	'': '',
	'': '',
	'': '',
	'': '',
}
```

```javascript
const obj = {
	'': '',
	'|': '',
	'': '',
	'': '',
	'': '',
}
```

```javascript
const obj = {
	'': '',
	'': '|',
	'': '',
	'': '',
	'': '',
}
```
